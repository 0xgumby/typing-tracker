# Typing Tracker

A lightweight Obsidian plugin that shows a temporary timestamp when you stop typing.
![7l56veyo7jce1-ezgif com-video-to-gif-converter](https://github.com/user-attachments/assets/5a874dd8-a791-4e8f-ac32-d2356f0cbf18)

## Features

- Configurable inactivity timeout
  
- RGB animation (my favorite), single color, and gradient color modes
- Optional Google Font or custom font URL support
<img width="676" height="510" alt="image" src="https://github.com/user-attachments/assets/42046048-f302-46a7-80af-ec748fcc50c9" />

## Development

1. Install dependencies:
   `npm install`
2. Build:
   `npm run build`
3. Copy these files into your vault plugin folder:
   - `manifest.json`
   - `main.js`
   - `styles.css`

## Community Plugin Release Files

For each release, attach/update:


- `manifest.json`
- `main.js`
- `styles.css`
